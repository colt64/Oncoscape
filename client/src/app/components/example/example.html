Example

To add a new analysis tool, follow these steps:

  1. Create a New Component
    • Copy the example component within apps/components/
    • Update the 3 files to match the new tool needs
    • Replace all "example" with the name of the new method
    • Update the HTML file to define the necessary user interface
    • Handle all stylistic changes within the <>.scss file
    • Change the <>.directive.js file for data and input handling
  2. Add the tool to the application menu
    • Within app/components/tools/tools.directive, include the name, route, image, and copy (text) describing the tool.
  3. Link the tool using the Rest API
    • Within app/index.route.js, add the state url, example, and authentication settings.

Add interactions with R through the R message handler. This can be accessed through the osApi variable in the AngularJS directive. Commands must be registered within app/component/oncoscape/oncoscape.api.service.js. To monitor the R console for debugging, set silent to false in the server/process.js file.

To use specific JS libraries (eg d3), include them in the constructor of the controller within the directive.
